# rogolabs.org

Domain redirect for **rogolabs.org** to [rogolabs.net](https://rogolabs.net/).

## How it works

This repository is served via GitHub Pages. The `index.html` file uses a
`<meta http-equiv="refresh">` tag to immediately redirect visitors from
`rogolabs.org` to `rogolabs.net`.

## Files

| File         | Purpose                                      |
| ------------ | -------------------------------------------- |
| `index.html` | HTML redirect to rogolabs.net                |
| `CNAME`      | Custom domain configuration for GitHub Pages |
